# Python SQLite Shop Management System

A simple command-line based shop management system developed using Python and SQLite.

## Features

* Customer registration
* SQLite database integration
* Product ordering system
* Payment method tracking
* Automatic order date storage
* Relational database structure using foreign keys

## Technologies Used

* Python
* SQLite3
* Datetime module

## Database Tables

### Customer Table

Stores:

* Customer ID
* Name
* Address
* Phone number
* Email

### Orders Table

Stores:

* Order ID
* Customer ID
* Payment method
* Ordered item
* Order date
* Total amount
